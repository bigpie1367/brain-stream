# Spec: 히스토리 테이블 너비 고정

## 문제
히스토리 탭에서 스크롤 시 각 행의 텍스트 길이(특히 곡 제목)에 따라 컬럼 너비가 동적으로 변하면서 테이블이 흔들리는 현상 발생.

## 원인
`table-layout: auto`가 적용되어 있어, `<th>`에 퍼센트 너비를 지정해도 브라우저가 콘텐츠 기반으로 너비를 재계산함.

## 해결
`table-layout: fixed`로 변경하면 `<th>`에 지정된 퍼센트 너비가 고정되어 콘텐츠와 무관하게 일정한 레이아웃 유지.

## 영향 범위
- `src/static/index.html` — CSS 변경 (table-layout, 컬럼 너비, auto-refresh 깜빡임 수정)
